## Deci

Simplified spreadsheet for you to add products and their specs / features to compare and make purchase DECIsion.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/T0mTG/deci.git
   ```
2. Navigate to the project folder:
   ```bash
   cd deci
   cd backend
   ```
3. Install dependencies:
   ```bash
   pip install -r .\requirements.txt
   ```
   
---

## Usage

Run the project along with the frontend
```bash
fastapi dev main.py
```

## Features
- Get/save data as JSON file locally 
- Save and load most recent JSON file
